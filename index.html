Firebase Authentication Check

This tool uses Firebase's fetchSignInMethodsForEmail method to check if an email is registered and which authentication methods are enabled.

fetchSignInMethodsForEmail(auth, email)
.then((methods) => {
  if (methods.length > 0) {
    // Email is registered
  } else {
    // Email is not registered
  }
});
